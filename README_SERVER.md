# دليل تشغيل سيرفر اللعبة (Backend)

هذا الدليل يشرح كيفية تشغيل السيرفر الخاص بـ `online_server.ts` على أي خادم (VPS) أو خدمة استضافة.

## 1. المتطلبات الأساسية
- Node.js (النسخة 18 أو أحدث).
- npm أو yarn.

## 2. خطوات التشغيل

1. **إعداد البيئة:**
   - انقل ملفات المشروع إلى الخادم.
   - تأكد من وجود ملفات المتغيرات البيئية (`.env`) في جذر المشروع وتحتوي على:
     ```env
     VITE_BACKEND_URL=ws://YOUR_SERVER_IP:YOUR_PORT
     PORT=YOUR_PORT
     ```

2. **تثبيت التبعيات:**
   - افتح الـ Terminal في مجلد المشروع ونفذ الأمر:
     ```bash
     npm install
     ```

3. **تشغيل السيرفر:**
   - يمكنك تشغيل السيرفر مباشرة باستخدام الأمر:
     ```bash
     npm start
     ```
   - أو إذا كنت تفضل إبقاء السيرفر يعمل في الخلفية (حتى بعد إغلاق الـ Terminal)، يُنصح باستخدام `pm2`:
     ```bash
     # تثبيت pm2
     npm install -g pm2
     
     # تشغيل السيرفر
     pm2 start npm --name "card-game-server" -- start
     
     # حفظ حالة السيرفر للبدء تلقائياً عند إعادة تشغيل الخادم
     pm2 save
     pm2 startup
     ```

## 3. ملاحظات هامة:
- تأكد من فتح المنفذ (Port) المستخدم (مثلاً `41561`) في جدار الحماية (Firewall) الخاص بالسيرفر.
- تأكد أن السيرفر يعمل ببروتوكول `ws://` وليس `wss://` (إلا إذا قمت بإعداد SSL شهادات تشفير على السيرفر).
